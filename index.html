JavaScript INTERVIEW QUESTIONS

Q1. What is JavaScript ?

JavaScript is a high-level, dynamic, untyped, and interpreted programming language that is widely used for web development to create interactive effects within web browsers.

Q2. Where do we insert JavaScript code ?

JavaScript code is inserted between <script> and </script> tags.

Q3. What is JavaScript Function ?

A JavaScript function is a block of JavaScript code, that can be executed when "called" for.
For Example: a function can be called when an event occurs, like when the user clicks a button.

Q4. Where should we insert JavaScript script tag ?

Scripts can be placed in the <body>, or in the <head> section of an HTML page, or in both.
If the script is placed in the <head> section, it will be executed before the page is displayed.
If the script is placed in the <body> section, it will be executed after the page is displayed.
